Most development environment setup is the same. Essentially install:

* VScode
* Rust
  * rust target
* _OSX_
  * _XCODE_
* Clone the repo
  
    ```sh
    git clone https://github.com/jbirbal-skydom/monolith.git
    cd monolith
    git submodule init
    git submodule update
    ```

* Build

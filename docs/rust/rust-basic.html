Welcome to our interactive Rust tutorial! This page demonstrates how to use Rust with live code examples.

Your First Rust Program​

Let's start with a simple "Hello, World!" program:

main.rs
1fn main() {
2  println!("Hello, World!");
3}

The program above demonstrates the basic structure of a Rust program. Let's break down what each part does:

  1. fn main() - This declares the main function, the entry point of our program
  2. println! - This is a macro that prints text to the console

Working with Variables​

Here's an example showing how to work with variables in Rust:

main.rs
1fn main() {
2  let x = 5;
3  let y = 10;
4  println!("x = {}, y = {}", x, y);
5  println!("sum = {}", x + y);
6}

Functions in Rust​

Let's explore how functions work in Rust:

main.rs
1fn add(a: i32, b: i32) -> i32 {
2  a + b  // Note: no semicolon means this is a return expression
3}
4
5fn main() {
6  let result = add(5, 3);
7  println!("5 + 3 = {}", result);
8}

Error Handling Example​

Here's how to handle potential errors in Rust:

main.rs
1fn divide(a: f64, b: f64) -> Result<f64, String> {
2  if b == 0.0 {
3      Err(String::from("Division by zero!"))
4  } else {
5      Ok(a / b)
6  }
7}
8
9fn main() {
10  match divide(10.0, 2.0) {
11      Ok(result) => println!("10 / 2 = {}", result),
12      Err(e) => println!("Error: {}", e),
13  }
14
15  match divide(10.0, 0.0) {
16      Ok(result) => println!("10 / 0 = {}", result),
17      Err(e) => println!("Error: {}", e),
18  }
19}

Image Generation Example​

Here's an example of generating a gradient image using Rust:

main.rs
1
2use image::{ImageBuffer, Rgb};
3use base64::Engine;  // Add this import
4
5fn main() {
6  // Create a 256x256 image with a simple gradient
7  let width = 256u32;
8  let height = 256u32;
9  let mut img = ImageBuffer::new(width, height);
10
11  // Fill with a purple to orange gradient
12  for (x, y, pixel) in img.enumerate_pixels_mut() {
13      let r = (x as f32 / width as f32 * 255.0) as u8;
14      let b = (y as f32 / height as f32 * 255.0) as u8;
15      *pixel = Rgb([r, 0, b]);
16  }
17
18  // Convert to PNG using a buffer
19  let mut png_data: Vec<u8> = Vec::new();
20  {
21      let mut cursor = std::io::Cursor::new(&mut png_data);
22      img.write_to(&mut cursor, image::ImageFormat::Png)
23          .expect("Failed to encode image");
24  }
25
26  // Encode to base64
27  let base64 = base64::engine::general_purpose::STANDARD.encode(&png_data);
28  println!("data:image/png;base64,{}", base64);
29
30}

Each example above is interactive - you can modify the code and run it directly in your browser!
